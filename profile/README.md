# 🧠 PSGEEKS Algorithm Study

꾸준히 알고리즘 문제를 해결하며
문제 해결 감각을 기르고, 사고의 유연성을 확장하는 것을 목표로 합니다.

단순 풀이가 아니라
접근 방식 · 사고 과정 · 리뷰 문화까지 함께 성장하는 스터디입니다.

<br />

## 🗓️ 진행 방식

### 0️⃣ 스터디 일정

	•	시작일: 2026-02-02
	•	매주 화요일까지 1문제 제출
	•	문제는 자율 선택
(프로그래머스, 백준, SWEA, 코드트리 등)

<br />


### 1️⃣ 문제 풀이 & PR 작성
	•	주 1회 문제를 풀고 PR 작성
	•	PR에는 다음 내용을 자유롭게 포함합니다:
		•	문제 접근 방식
		•	풀이 전략
		•	사고 흐름
		•	시행착오
		•	배운 점
	•	주차별 브랜치를 생성하여 작업
	•	PR은 main 브랜치로 머지

<br />


### 2️⃣ 리뷰 문화
	•	다음 제출일까지 다른 인원의 문제를 풀고 리뷰 작성
	•	화요일이 되면 모든 PR은 closed 상태여야 합니다.
	•	리뷰는 approved로 진행
	•	모든 인원이 리뷰 완료 시 merge

<br />
